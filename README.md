# Spec-MarkDown

本质是为了目前在写的 pgcm 社区所写的markdown 编辑器

```shell
# npm
npm install spec-markdown-editor

# pnpm
pnpm install spec-markdown-editor
```

例:

```jsx
import SpecMarkdown from 'spec-markdown-editor'

function App() {
    return (
        <>
            <SpecMarkdown />
        </>
    )
}
```

## 基本功能

- [x] 实时预览
- [x] 同步滚动
- [ ] 主题
- [ ] 快捷键
- [ ] 工具栏
- [ ] 图片
- [ ] 导入/ 导出
